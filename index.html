Scrolly Telling With D3

Below is a simple example of using updatable and interactive charts for "scrolly telling." I developed this in a few hours using only D3, TypeScript, and no other libraries. Scroll down for the demo.

This Section Goes With The Bar Chart

The bar chart on the right is a simple example of an updatable D3.js visualization. The user's scrolling behavior controls this chart's actions through update methods. These methods can change the height and color of the bars or modify the chart's data. While this section of text is visible, scrolling controls the chart's height.

This Section Changes the Bar Chart's Data

When entered, each section's controller updates the chart's data and the color of the bars. If you entered this section from above, the section controller updated the chart's data and color. If you entered this section from below, the current section's controller initialized a new bar chart and attached it to the SVG on the right, and the controller from the section above will modify this chart. If you leave this section by scrolling up, the section controller will clean up and remove the bar chart.
